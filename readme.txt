基于SpringBoot的微信点餐系统后台开发
项目概要：为买家（微信端）和卖家（PC端）提供一款外卖点餐平台
功能分析：商品管理 / 订单管理 / 微信支付与退款 / 接单提醒 / 商品秒杀 / 账号登陆
实现过程：1.利用MySQL数据库作为本地存储工具；2.利用MyBatis实现对数据库的操作并实现商品增改、商品查询、新建订单等方法；
3.利用一款开源的微信支付SDK实现订单支付与退款；4.卖家后端页面采用BootStrap实现；
5.利用WebSocket协议实现卖家端的接单提醒；6.利用Redis实现分布式锁，解决商品秒杀时多线程高并发问题；
7.利用Redis+session实现卖家账号的登陆。
